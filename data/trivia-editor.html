How to review questions

  1. Click any question card to expand it and check the answer, options, and explanation.
  2. Edit any field directly — changes auto-save for everyone.
  3. When a question looks good, click the READY toggle to mark it ready for users.
  4. Only questions marked READY will appear in the app.

Controls

  • Arrow buttons — reorder questions
  • Pencil — open full edit modal
  • X — delete a question
  • + Add Question — create a new question from scratch
  • Save JSON — download a local copy of all questions

How saving works

Every change auto-saves to GitHub within 2 seconds. The green Synced badge in the header confirms your changes are live. If you see Sync failed, try refreshing the page.
